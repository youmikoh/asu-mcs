set default list indent size using only default packages

To set the default list indent size using only default packages, you can use the enumitem package, which is included in most LaTeX distributions. The enumitem package allows you to customize the appearance of lists, including indentation.

Here's an example of how to set the default list indent size using the enumitem package:

latex
\documentclass{article}
\usepackage{enumitem}

% Set default list indent size
\setlist{
  leftmargin=*,
  labelindent=0.5cm,
  itemindent=0.5cm
}

\begin{document}

\begin{enumerate}
    \item First item
    \item Second item
    \begin{enumerate}
        \item Nested item 1
        \item Nested item 2
    \end{enumerate}
    \item Third item
\end{enumerate}

\end{document}

In this example, the leftmargin=* option ensures that the list is aligned with the surrounding text, while the labelindent and itemindent options set the indent size for the list labels and items, respectively. Adjust the values of labelindent and itemindent to customize the indent size to your preference.

To remove table margins and padding in LaTeX, you can use the array package, which allows you to customize the appearance of tables. You can adjust the \arraystretch and \tabcolsep parameters to control the row height and column separation, respectively.

latex

\begin{document}

\hline
Cell 1 & Cell 2 \\
\hline
Cell 3 & Cell 4 \\
\hline
\end{tabular}

\end{document}

In this example, the \arraystretch parameter is set to 1, which is the default value, and the \tabcolsep parameter is set to 0pt, which removes the padding between columns. Adjust the values of \arraystretch and \tabcolsep to customize the table margins and padding to your preference.
